# Table Export Guide

更新时间：2026-06-09

当前导出策略是前端当前页 CSV 导出。

## 当前策略

- 只导出当前表格页。
- 由 Antdpro6 `TableExportButton` 在浏览器生成 CSV。
- 空数据时显示提示，不生成文件。
- 文件名由页面传入，通常包含模块名和当前 tab/scope。

## 不做什么

- 不做导入。
- 不做后端全量同步导出。
- 不做后端异步导出任务。
- 不做超大数据量导出。

大数据量导出未来应另开专项，设计后端异步任务、文件中心落盘、下载权限和过期策略。

## 当前覆盖

- 用户管理
- 角色管理
- 权限管理
- 菜单管理
- 图片资源
- 字典管理
- 系统参数
- 文件中心
- 操作日志
- 登录日志
- 消息中心
- 审批请求

## 新页面接入

1. 在页面保存当前请求返回的行数据。
2. 在 `toolBarRender` 中加入 `TableExportButton`。
3. 明确 `filename`。
4. 明确 `columns`，不要直接导出复杂对象。
5. E2E 至少覆盖按钮存在；关键业务页覆盖下载文件名。
